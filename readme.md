# Sobre a estrutura do teste:

Pasta *assets*:
  - SCSS
  - HTML
  - JS
  - Sprites
  
Pasta *build*:
  - HTML Gzipado
  - CSS Gzipado
  - JS Minificado

O Gulpfile.js desse teste é baseado no do generator webapp do Yeoman, porém bastante modificado. Para visualizar o resultado, digite a seguinte linha de comando no CMD/Terminal:

```sh
$ npm install
$ gulp
```
